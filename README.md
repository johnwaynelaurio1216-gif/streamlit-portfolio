# streamlit-portfolio
Streamlit Multipage Web Application
Project Description

This project is a multipage web application developed using Streamlit in Python.
It serves as a simple personal portfolio that displays information about myself, including my background, skills, projects, and contact details.

The purpose of this project is to demonstrate the ability to build a basic web application with a clean interface and multiple pages using Streamlit.

Objectives
Create a multipage web application using Streamlit
Apply basic UI/UX design principles
Add simple interactive elements such as inputs and buttons
Present personal portfolio information in an organized way
Pages
Home – Introduction to the portfolio
About – Personal background and education
Skills – Technical skills
Projects – Sample works and outputs
Contact – Message form for communication
Features
Sidebar navigation for multiple pages
Clean and simple user interface
Interactive input components (text fields, buttons, sliders)
Organized page structure
Tools Used
Python
Streamlit
How to Run the Application

Install Streamlit:

pip install streamlit

Run the application:

streamlit run Home.py
